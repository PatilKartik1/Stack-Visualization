What is a Stack?

A stack is a linear data structure that follows the Last In, First Out (LIFO) principle. The last element added to the stack is the first one to be removed — like a stack of plates where you always take from the top.

Push Add an element to the top of the stack.
Pop Remove and return the top element of the stack.
Peek View the top element without removing it.
